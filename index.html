Zacznij teraz

Zamień nudne commity w fascynujący blog! ✨

git2blog automatycznie przekształca historię Git w profesjonalny blog używając lokalnego AI. Żadnych serwerów, żadnych kosztów - tylko czysta magia! 🪄

Rozpocznij za darmo Zobacz demo
# Zainstaluj git2blog
pip install git2blog
git2blog --init
git2blog
# 🎉 Twój blog jest gotowy w ./blog/

Dlaczego git2blog? 🤔

Przestań tracić czas na nudne pisanie postów. Niech AI zrobi to za Ciebie, bazując na tym co już robisz - commitach Git!

🤖

AI-Powered Content

Używa lokalnego Ollama AI do generowania angażujących postów z suchych commitów. Żadnych API keys, żadnych limitów!

⚡

Błyskawicznie szybkie

Wygeneruj kompletny blog w sekundach. Jeden commit = jeden post. Automatycznie, inteligentnie, efektywnie.

🎨

Piękny design

Nowoczesne, responsywne szablony HTML. Dark mode, animacje, mobile-first. Gotowe do publikacji!

🔒

100% prywatne

Wszystko działa lokalnie. Twój kod nigdy nie opuszcza komputera. Zero trackingu, zero wysyłania danych.

⚙️

Super konfigurowalne

Dostosuj wszystko: modele AI, style CSS, prompty, filtry commitów. Pełna kontrola nad treścią.

🐳

Docker ready

Gotowe obrazy Docker z Ollama. Deploy jedną komendą. CI/CD friendly i DevOps approved!

Zobacz git2blog w akcji! 🎬

Od commitów do gotowego bloga w 3 prostych krokach

1️⃣

Zainstaluj & Skonfiguruj

pip install git2blog
ollama pull llama3.2
git2blog --init
2️⃣

Dostosuj konfigurację

# git2blog.yaml
blog_title: 'Mój Dev Blog'
model: 'llama3.2'
commit_limit: 20
3️⃣

Wygeneruj blog!

git2blog
# ✨ Blog gotowy w ./blog/
open blog/index.html
Spróbuj teraz za darmo!

Gotowy na AI-powered blogging? 🚀

Dołącz do setek deweloperów którzy już używają git2blog do tworzenia angażujących treści z ich codziennej pracy!

curl -fsSL https://ollama.ai/install.sh | sh
